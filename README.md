# Breaking Spears

Breaking Spears is a Fabric mod for Minecraft `1.21.11` that adds a fast, cinematic way to fly through obstacles with Elytra.

## Features

- While gliding with Elytra and actively using a spear or trident, colliding with a glass pane breaks the pane so you can continue flying through.
- Adds enchantment `Breaking` (`Razrushitel`) for tridents, levels `I-III`.
- Thrown trident with `Breaking` can destroy one block per throw (level-based drop chance and trident durability loss on successful break).
- Blocks destroyed by thrown trident are limited to one per throw (no chain destruction).

## Version

Current mod version: `1.1.0`.
